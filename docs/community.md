# Community

TchueKAM is built by people who care, for people who care. Here's how to plug in.

## Discussions and ideas
The best place to share feature requests, ask questions, and see what others are building:

**[github.com/Tchuekam/tchuekam-desktop/discussions](https://github.com/Tchuekam/tchuekam-desktop/discussions)**

## Report a bug or contribute
**[github.com/Tchuekam/tchuekam-desktop](https://github.com/Tchuekam/tchuekam-desktop)** — file issues, browse code, submit pull requests.

## Stay updated
- Star the repo to get release notifications
- Follow the [website](https://tchuekam.com) for major announcements

## Code of conduct
We expect everyone in the TchueKAM community to be respectful and constructive. Disagreement is fine; disrespect isn't. Violations may result in removal from community spaces.

## Get involved
- **Translate** — help bring TchueKAM to more languages
- **Write docs** — improve these pages (every doc page has an "Edit this page" link)
- **Build features** — see good-first-issue tags on GitHub
- **Spread the word** — tell a friend, write a blog post, share what you've built

We notice. Thank you.
