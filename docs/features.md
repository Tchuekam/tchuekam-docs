# Features

What TchueKAM does, organized by what you'd want to do with it.

## Conversation
- **Natural chat** — ask anything, get answers grounded in your context
- **Streaming responses** — see TchueKAM think in real time
- **Voice input** — push-to-talk for hands-free interaction
- **Conversation history** — searchable, organized by project

## Files and workspace
- **Workspace awareness** — TchueKAM understands the folder you're working in
- **File reading and editing** — open, summarize, edit any file
- **Terminal integration** — run commands and read output safely
- **Project memory** — remembers context across sessions per workspace

## Code
- **Multi-language code generation** — Python, JavaScript, TypeScript, Go, Rust, and more
- **Inline edits** — surgical changes to specific lines
- **Test scaffolding** — generate test files from existing code
- **Git-aware** — knows your branch, recent changes, and history

## Web and external
- **Web search** — pull in current information when needed
- **URL fetching** — read and summarize web pages
- **Image understanding** — describe screenshots, extract text from images

## System integration
- **Native menus** — feels like a real desktop app, not a browser
- **System notifications** — get notified when long tasks finish
- **Clipboard integration** — paste text, images, or files directly
- **Keyboard shortcuts** — fully customizable

## Privacy and control
- **Local-first** — your conversations never leave your machine unless you choose
- **Configurable data sharing** — opt in or out per feature
- **Export anything** — download conversations as Markdown or JSON
- **Delete anything** — full data removal in one click

For exhaustive technical capabilities, see [AI Capabilities](ai-capabilities).
