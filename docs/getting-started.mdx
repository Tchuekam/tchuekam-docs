# Getting Started

Get TchueKAM Agent running on your computer in under five minutes.

## 1. Download

<LatestRelease />

The download starts immediately. Save the installer somewhere you can find it (your Downloads folder is fine).

## 2. Install

### Windows
Double-click `TchuekaM-Setup-win-x64.exe`. The installer will:
- Ask where to install TchueKAM (default: `%LocalAppData%\Programs\TchueKAM`)
- Create a desktop shortcut and Start menu entry
- Take about 30 seconds

### macOS
Open `TchuekaM-Setup-mac-arm64.dmg` (or `-x64.dmg` on Intel Macs). Drag **TchueKAM** to **Applications**. The first launch will ask permission to run an app downloaded from the internet — click **Open**.

### Linux
The AppImage is portable: download, `chmod +x`, double-click to run. Or place it in `~/.local/bin/` to make it available everywhere.

## 3. Launch

Open TchueKAM the same way you launch any other app. On first launch, TchueKAM will:

1. Run a short one-time setup (about 15–30 seconds)
2. Ask for microphone permission if you want voice input (optional, you can grant later)
3. Show you the main workspace

## 4. Start a conversation

Type anything in the chat box and press **Enter**. TchueKAM will respond. Try:

- _"Summarize the README in this folder."_
- _"Generate a Python script that scans my Downloads for duplicates."_
- _"What did I work on last week?"_

## 5. You're done

TchueKAM is fully configured. Updates install automatically — you'll see a notification when a new version is ready.
